About the tool

Semoss is an innovative and data-driven application that allows users to explore and uncover connections among existing data from multiple repositories in an effort to help make more informed decisions.
The tool displays data from one or more databases in an interactive format that users can customize, overlay, and extend based on their individual needs. This can help establish connections between the pieces of data that healthcare providers need and those system(s) that provide it.

As for the name, the word Semoss is a play on the Sea moss, which looks a lot like the a network graph with lots of nodes. It also stands for Semantic Operating Source Software

Why we made it

Semoss was originally built to:
- Help consolidate system information into a central location
- Identify ripple effects associated with interface changes
- Assist Program Managers with planning and migration efforts
- Support development of plans/estimates for modernizing systems to a net-centric environment based on Service Oriented Architecture (SOA)
- Identify systems that support specific capabilities
- Reduce portfolio duplication by finding redundancies in data/system functions
- Support alignment with Integrated Master Schedules (IMS)
- Help identify interim architectures based on functional needs
