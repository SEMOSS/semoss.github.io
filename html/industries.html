  • Discover unique robust solutions with data analytics across the government

    Aiding researchers to discover treatments for life-threatening infectious diseases

    Faced with the urgency of finding treatments for infectious diseases, such as Ebola, researchers at the National Institute of Allergy and Infectious Diseases (NIAID) turn to SEMOSS to find, access, visualize, and share relevant data.

    Due to significant investment and federal mandate in open data initiatives, data has never been more abundant. Unfortunately, data access remains a challenge due to myriad data providers and data formats. These are prevalent issues for researchers who do not have the time to assemble data into a manageable form for analysis. Often, finding and linking data is such a hassle that researchers instead opt for unauthoritative datasets or recreate existing datasets, exhausting limited grant funds and reducing the speed of scientific discovery.

    Helping researchers combine open data

    Data preparation tops the list among data difficulties. Analysis features are essential. And as an open source platform, SEMOSS can provide them all. However, the true problem plaguing the research community is data access – the data was never created to be combined, so researches spend an inordinate amount of their time integrating data. This is why SEMOSS is first and foremost a simple user interface to find and access relevant data.

    Similar to the way travel search engines pull data from hotel and airport websites, SEMOSS seamlessly integrates third-party data without the trouble and expense of full data integration or data warehouse creation. Using semantic web technologies (including W3C standards such as the RDF model for data interchange on the web), SEMOSS provides a single platform that allows researchers to access data from multiple providers with a point and click interface.

    Accelerating Scientific Insight

    SEMOSS provides the ease of use to accelerate the scientific process. Using the D3.js JavaScript library, SEMOSS providing intuitive interactive visualizations – ideal for both directed and undirected research. Furthermore, collaboration has never been easier, as SEMOSS provides simple social sharing to send interactive visualizations to collaborate with colleagues.
