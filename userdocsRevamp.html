User Documentation

Getting up and running with your analytics enviroment

Introduction

SEMOSS is a data-driven application that enables users to explore relationships and connections in an interactive format through intermediate conduits, i.e. transitive relationships. This document provides general usage instructions and guidance in deployment of the open source components of SEMOSS, in contrast to the developer guide, which provides instructions and guidance in terms of development "on top of" the tool.

Overview

    This document contains:
  • An overview of the out-of-the-box SEMOSS user interface (Section 1).
  • Instructions for creating and loading a new database and developing the ontological structure (metamodel) for the database (Section 2).
  • Limited guidance in querying databases to understand advanced relationships using the SPARQL query language (Section 3).
  • A use case in deploying “out of the box” SEMOSS to analyze the sample security database (Section 4).
